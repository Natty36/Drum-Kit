# Drum ü•Å Kit

An interactive drum kit website built using HTML, CSS, and JavaScript.  
The interactive website allows users to play different drum sounds by clicking on on-screen buttons or pressing the corresponding keys on the keyboard.

I created this project was while learning JavaScript and DOM manipulation as part of a full-stack web development course.

## Features
- Play drum sounds using mouse clicks
- Keyboard support for triggering sounds
- Button animation when a drum is played
- Simple layout that adapts naturally to different screen sizes

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation & event handling)

## What I Learned
- Adding event listeners for mouse and keyboard events
- Using the `this` keyword and event objects
- understanding the difference between regular functions and arrow functions
- Playing audio in a web application
- Working with JavaScript functions, callback functions, and objects
- Adding basic animations using JavaScript

## How to Use
1. Open the website in your browser
2. Click on any drum button or press the matching keyboard key
